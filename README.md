# Gamesta

Gamesta is a front end project developed as part of a Webermelon internship task. The project focuses on layout precision, with particular attention to CSS positioning techniques.

## Overview

This project highlights the ability to build structured and visually accurate user interfaces based on given requirements. It demonstrates effective use of CSS positioning, especially absolute positioning, to control layout and element placement.
